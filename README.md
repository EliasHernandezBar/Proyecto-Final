# Proyecto-Final
Proyecto Final Bootcamp — Data Analytics &amp; Machine Learning con PostgreSQL, Power BI y Python
Proyecto integral de análisis de datos desarrollado como trabajo final de bootcamp, enfocado en la construcción de una arquitectura analítica completa: desde el modelado de datos y generación de KPIs hasta visualización en Power BI y preparación de datasets para Machine Learning en Python.


🛠️ Tecnologías utilizadas
. PostgreSQL
. DBeaver
. SQL
. Power BI
. Python
. Pandas
. Scikit-learn
